## プロダクト概要
- プロダクト名:
  コミュフェイス
- コンセプト:
  表情と小ストーリーで笑顔の輪を広げる
- 説明:
  コミュフェイスは出来事をすぐ書き出し小ストーリーにし、表情を添えることであなたの新鮮な感情をカードにまとめ友人に共有できます
- 推しポイント：
  出力されたストーリーカードがいい
## 開発技術
- 利用したプログラミング言語 : Kotlin
- 利用したフレームワーク・ライブラリ
- その他開発に使用したツール : Android studio
